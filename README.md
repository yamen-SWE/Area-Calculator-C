# Area Calculation (C Language)

## Description
A practical program written in C.

This application calculates the areas of simple geometric shapes, demonstrating sound logic and structured programming in C.

## Features
* **Geometric Shapes:** Calculates the areas of squares, rectangles, and triangles.

* **Language:** C.

* **Integrated Development Environment:** Code::Blocks.

* **Input Validation:** Ensures that user input is processed correctly for accurate results.

* **Organized Repository:** Uses the `.gitignore` file to keep the project free of unnecessary IDE files.

## How to Run
1. Clone the repository.

2. Open the `Area_Calculation.cbp` file in Code::Blocks.

3. Build and run the program.
