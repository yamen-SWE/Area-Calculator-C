# C Geometry Area Calculator 📐

## Description
A practical and interactive command-line program written in C that calculates the area of various geometric shapes. The application runs continuously in a loop, demonstrating sound dynamic logic, structured programming, and clean error handling.

## 🚀 Features
* **Expanded Geometric Shapes:** Calculates the precise areas of:
  - **Square** ($Area = s^2$)
  - **Rectangle** ($Area = l \times w$)
  - **Triangle** ($Area = 0.5 \times b \times h$)
  - **Circle** ($Area = \pi \times r^2$) using `<math.h>` library functions.
  - **Trapezoid** ($Area = 0.5 \times (b1 + b2) \times h$) supporting multiple inputs in a single line.

* **Continuous Execution:** Implements a user-controlled loop that keeps the application active until explicitly terminated by entering `n` or `N`.

* **Robust Logic & Math:** Employs precise float division `(22/7.0)` to maintain high fractional accuracy for $\pi$ (Pi).

* **Input Validation:** Features a `default` safety case within the switch-case architecture to handle invalid menu numbers gracefully.

* **Organized Repository:** Utilizes a `.gitignore` file to ensure the repository remains clean and free of unnecessary IDE build files.

## ⚙️ How to Run
1. Clone the repository.
2. Open the project file in **Code::Blocks** or any modern C compiler.
3. Build and run the `main.c` file.
4. Follow the intuitive on-screen menu instructions.
